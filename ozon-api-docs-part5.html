Получить отчёт по товарам в оплате за заказ — выбранные товары

POST /api/client/statistic/products/generate

В параметрах запроса from и to укажите промежуток времени.

Performance API асинхронный. Поэтому в результате запроса будет не сам отчёт, а уникальный идентификатор отправленного запроса, с помощью которого можно проверить статус формирования отчёта и скачать отчёт в формате CSV.

Чтобы запросить отчёт в формате JSON, добавьте /json в адрес метода: /api/client/statistic/products/generate/json.

Отчёт содержит:

  • период отчёта;
  • SKU;
  • артикул;
  • наименование;
  • категорию;
  • продвижение;
  • цену товара, ₽;
  • ставку, %;
  • ставку, ₽;
  • количество заказов;
  • сумму заказов, ₽;
  • расход, ₽;
  • продажи («Оплата за клик»), ₽;
  • расход («Оплата за клик»), ₽;
  • заказы («Оплата за клик»), шт;
  • в корзину;
  • ДРР, %;
  • последнее изменение.
REQUEST BODY SCHEMA: application/json
from
string <date-time>

Начальная дата периода отчёта в формате RFC 3339.

to
string <date-time>

Конечная дата периода отчёта в формате RFC 3339.

Ответы

200Статистика по кампании∨

Уникальный идентификатор отправленного запроса.
По нему можно проверить статус формирования отчёта и скачать отчёт.
vendor
boolean
Если запрашивается отчёт с аналитикой внешнего трафика — true.
∨defaultОшибка запроса
RESPONSE SCHEMA:application/json
