Статистика по расходу кампаний

GET /api/client/statistics/expense

Если в запросе не указаны значения полей dateFrom и dateTo, то в ответе будет статистика за последние 7 дней.

Формат отчёта — CSV.

Чтобы запросить отчёт в формате JSON, добавьте в адрес метода /json и отправьте запрос по адресу /api/client/statistics/expense/json.

Отчёт содержит:

  • идентификатор кампании,
  • дата,
  • название,
  • расход,
  • расход с абонентского счёта,
  • расход бонусов.
QUERY PARAMETERS
campaignIds
Array of strings <uint64>

Список идентификаторов кампаний, для которых необходимо подготовить отчёт.

Если поле заполнено, в отчёте будет статистика только по указанным кампаниям. Если поле пустое, в ответе будет статистика по всем кампаниям в указанном промежутке времени.

Необязательное поле.

from
string <date-time>

Начальная дата периода отчёта в формате RFC3339.

Необязательное поле.

to
string <date-time>

Конечная дата периода отчёта в формате RFC3339.

Необязательное поле.

dateFrom
string

Начальная дата периода отчёта в формате ГГГГ-ММ-ДД. Например: 2019-02-10.

Необязательное поле.

dateTo
string

Начальная дата периода отчёта в формате ГГГГ-ММ-ДД. Например: 2019-02-10.

Необязательное поле.
