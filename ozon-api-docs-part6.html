Отчёт по поисковым запросам

POST /api/client/statistics/phrases
Метод находится на стадии тестирования.

Отчёт можно сформировать только для кампаний с оплатой за клик с признаком placement = PLACEMENT_TOP_PROMOTION.

Чтобы запросить отчёт в формате JSON, добавьте /json в адрес метода и отправьте запрос по адресу /api/client/statistics/phrases/json.

REQUEST BODY SCHEMA: application/json
campaigns
Array of strings <uint64>

Список кампаний.

dateFrom
string

Начальная дата периода отчёта в формате ГГГГ-ММ-ДД. Например: 2019-02-10. Дата не может быть раньше 1 февраля 2025 года.

dateTo
string

Конечная дата периода отчёта в формате ГГГГ-ММ-ДД. Например: 2019-02-10.

from
string <date-time>

Начальная дата периода отчёта в формате RFC 3339. Дата не может быть раньше 1 февраля 2025 года.

to
string <date-time>

Конечная дата периода отчёта в формате RFC 3339.

groupBy
string (extstatisticsGroupBy)

Default: "NO_GROUP_BY"

Enum: "NO_GROUP_BY" "DATE" "START_OF_WEEK" "START_OF_MONTH"

Тип группировки по времени:

  • DATE — группировка по дате (по дням);
  • START_OF_WEEK — группировка по неделям;
  • START_OF_MONTH — группировка по месяцам.

Ответы

200Отчёт по поисковым запросам∨
RESPONSE SCHEMA:application/json
UUID
string
Уникальный идентификатор отправленного запроса.
По нему можно проверить статус формирования отчёта и скачать отчёт.
vendor
boolean
Если запрашивается отчёт с аналитикой внешнего трафика — true.
∨defaultОшибка
RESPONSE SCHEMA:application/json
error
string
Описание ошибки.

Примеры запроса

Content type

{
  "campaigns": [
    "string"
  ],
  "dateFrom": "string",
  "dateTo": "string",
  "from": "2019-08-24T14:15:22Z",
  "to": "2019-08-24T14:15:22Z",
  "groupBy": "NO_GROUP_BY"
}

Оплата за клик

Создать кампанию с оплатой за клики

POST /api/client/campaign/cpc/v2/product

Метод для создания товарной рекламной кампании с оплатой за клики.

При успешном выполнении запроса в поле campaignId ответа будет идентификатор созданной кампании.

REQUEST BODY SCHEMA: application/json
title
string

Название рекламной кампании.

fromDate
string

Дата начала рекламной кампании по московскому времени.

Если параметр не заполнен, датой старта считается начало текущего дня. Открутка начинается сразу после активации кампании, если не нужна модерация.

toDate
string

Дата окончания рекламной кампании по московскому времени. Параметр не учитывается для кампаний с оплатой за клики, созданных в автоматическом режиме.

dailyBudget
string <uint64>
Deprecated

Ограничение дневного бюджета рекламной кампании. Единица измерения — одна миллионная доля рубля, округляется до копеек. Например, значение 1 000 000 в параметре равно 1 рублю.

Если параметр не заполнен, дневной бюджет не ограничен. После создания рекламной кампании изменить бюджет с дневного на недельный и наоборот не получится.

weeklyBudget
string <uint64>

Ограничение недельного бюджета рекламной кампании. Единица измерения — одна миллионная доля рубля, округляется до копеек. Например, значение 1 000 000 в параметре равно 1 рублю.

Если параметр не заполнен, недельный бюджет не ограничен. После создания рекламной кампании изменить бюджет с недельного на дневной и наоборот не получится.

placement required
string (string)

Default: "PLACEMENT_INVALID"

Enum: "PLACEMENT_TOP_PROMOTION" "PLACEMENT_INVALID" "PLACEMENT_SEARCH_AND_CATEGORY"

Место размещения продвигаемых товаров:

  • PLACEMENT_TOP_PROMOTION — поиск;
  • PLACEMENT_INVALID — не определено;
  • PLACEMENT_SEARCH_AND_CATEGORY — поиск и рекомендации.
productAutopilotStrategy
string (string)

Enum: "MAX_CLICKS" "TOP_MAX_CLICKS" "TARGET_BIDS" "TOP_PROMOTION" "TARGET_CIR"

Стратегия, которая будет использоваться для кампании:

  • MAX_CLICKS — автостратегия для Поиска и рекомендаций;
  • TOP_MAX_CLICKS — автостратегия для Поиска;
  • TARGET_BIDS — средняя стоимость клика для Поиска и рекомендаций;
  • TOP_PROMOTION — вывод в топ;
  • TARGET_CIR — целевой расход.

Ответы

200Кампания создана∨
RESPONSE SCHEMA:application/json
campaignId
string
Идентификатор созданной кампании.
∨defaultОшибка
RESPONSE SCHEMA:application/json
error
string
Описание ошибки.

  "productAutopilotStrategy": "MAX_CLICKS"
