Отчёт по поисковым запросам

POST /api/client/statistics/phrases
Метод находится на стадии тестирования.

Отчёт можно сформировать только для кампаний с оплатой за клик с признаком placement = PLACEMENT_TOP_PROMOTION.

Чтобы запросить отчёт в формате JSON, добавьте /json в адрес метода и отправьте запрос по адресу /api/client/statistics/phrases/json.

REQUEST BODY SCHEMA: application/json
campaigns
Array of strings <uint64>

Список кампаний.

dateFrom
string

Начальная дата периода отчёта в формате ГГГГ-ММ-ДД. Например: 2019-02-10. Дата не может быть раньше 1 февраля 2025 года.

dateTo
string

Конечная дата периода отчёта в формате ГГГГ-ММ-ДД. Например: 2019-02-10.

from
string <date-time>

Начальная дата периода отчёта в формате RFC 3339. Дата не может быть раньше 1 февраля 2025 года.

to
string <date-time>

Конечная дата периода отчёта в формате RFC 3339.

groupBy
string (extstatisticsGroupBy)

Default: "NO_GROUP_BY"

Enum: "NO_GROUP_BY" "DATE" "START_OF_WEEK" "START_OF_MONTH"

Тип группировки по времени:

  • DATE — группировка по дате (по дням);
  • START_OF_WEEK — группировка по неделям;
  • START_OF_MONTH — группировка по месяцам.

Ответы

∨defaultОшибка
