Рассчитать минимальный бюджет кампании Deprecated

POST /external/api/dynamic_budget
1 сентября 2025 года метод будет отключён. Теперь минимальный бюджет рассчитывается по формуле: 2 000 рублей × 1 SKU.
REQUEST BODY SCHEMA: application/json
createCampaign ›
object (object)

Параметры для расчёта при создании кампании.

updateCampaign ›
object (object)

Параметры для расчёта при обновлении кампании.

Ответы

200Результат запроса∨
RESPONSE SCHEMA:application/json
campaignBudget
string <uint64>
Бюджет рекламной кампании. Единица измерения — одна миллионная доля рубля, округляется до копеек. Например, значение 1 000 000 в параметре равно 1 рублю.
dynamicBudget
string <uint64>
Минимальный бюджет.
dynamicBudgetType
string (string)
Default: "DYNAMIC_BUDGET_REQUIRED"
Enum: "DYNAMIC_BUDGET_REQUIRED" "DYNAMIC_BUDGET_RECOMMENDED"
Тип минимального бюджета:
  • DYNAMIC_BUDGET_REQUIRED — обязательный;
  • DYNAMIC_BUDGET_RECOMMENDED — рекомендованный.
skuPrice
string <uint64>
Сумма в рублях, на которую увеличивается минимальный бюджет на каждый SKU.
∨defaultОшибка
RESPONSE SCHEMA:application/json
error
string
Описание ошибки.
