Выключить кампанию

POST /api/client/campaign/{campaignId}/deactivate
PATH PARAMETERS
campaignId required
string <uint64>

Идентификатор кампании.

REQUEST BODY SCHEMA: application/json
object (Empty)

Ответы

200Кампания деактивирована∨
RESPONSE SCHEMA:application/json
id
string <uint64>
Идентификатор кампании.
autostopStatus
string (advcamptypeAutostopStatus)
Default: "AUTOSTOP_STATUS_NONE"
Enum: "AUTOSTOP_STATUS_NONE" "AUTOSTOP_STATUS_ENABLED" "AUTOSTOP_STATUS_DISABLED" "AUTOSTOP_STATUS_STOPPED"
Статус кампании:
  • AUTOSTOP_STATUS_NONE — продавец создал кампанию вручную;
  • AUTOSTOP_STATUS_ENABLED — кампания создана автоматически и остановится, когда кончатся баллы;
  • AUTOSTOP_STATUS_DISABLED — кампания создана автоматически, но она не остановится, когда кончатся баллы;
  • AUTOSTOP_STATUS_STOPPED — баллы закончились, кампания остановлена автоматически.
isAutocreated
boolean
true, если кампания создана автоматически.
paymentType
string (string)
Default: "CAMPAIGN_TYPE_INVALID"
Тип оплаты:
  • CPC — за клики.
title
string
Название кампании.
state
string (string)
Default: "CAMPAIGN_STATE_UNKNOWN"
Состояние кампании:
  • CAMPAIGN_STATE_RUNNING — активная кампания;
  • CAMPAIGN_STATE_PLANNED — кампания, сроки проведения которой ещё не наступили;
  • CAMPAIGN_STATE_STOPPED — кампания, приостановленная из-за нехватки бюджета;
  • CAMPAIGN_STATE_INACTIVE — кампания, остановленная владельцем;
  • CAMPAIGN_STATE_ARCHIVED — архивная кампания;
  • CAMPAIGN_STATE_MODERATION_DRAFT — отредактированная кампания до отправки на модерацию;
  • CAMPAIGN_STATE_MODERATION_IN_PROGRESS — кампания, отправленная на модерацию;
  • CAMPAIGN_STATE_MODERATION_FAILED — кампания, непрошедшая модерацию;
  • CAMPAIGN_STATE_FINISHED — кампания завершена, дата окончания в прошлом, такую кампанию нельзя изменить, можно только клонировать или создать новую.
fromDate
string
Дата старта рекламной кампании.
toDate
string
Дата завершения рекламной кампании.
budget
string <uint64>
Бюджет рекламной кампании. Единица измерения — одна миллионная доля рубля, округляется до копеек. Например, значение 1 000 000 в параметре равно 1 рублю.
dailyBudget
string <uint64>
Deprecated
Дневной бюджет рекламной кампании. Единица измерения — одна миллионная доля рубля, округляется до копеек. Например, значение 1 000 000 в параметре равно 1 рублю.
weeklyBudget
string <uint64>
Недельный бюджет рекламной кампании. Единица измерения — одна миллионная доля рубля, округляется до копеек. Например, значение 1 000 000 в параметре равно 1 рублю.
placement
string (extcampaignProductCampaignPlacement)
Default: "PLACEMENT_INVALID"
Enum: "PLACEMENT_INVALID" "PLACEMENT_PDP" "PLACEMENT_SEARCH_AND_CATEGORY" "PLACEMENT_TOP_PROMOTION" "PLACEMENT_OVERTOP"
Место размещения продвигаемых товаров:
  • PLACEMENT_INVALID — не определено;
  • PLACEMENT_PDP — карточка товара. Доступно только для кампаний с ручным управлением;
  • PLACEMENT_SEARCH_AND_CATEGORY — поиск и рекомендации;
  • PLACEMENT_TOP_PROMOTION — поиск;
  • PLACEMENT_OVERTOP — поиск.
productAutopilotStrategy
string
Enum: "MAX_CLICKS" "TOP_MAX_CLICKS" "TARGET_BIDS" "TOP_PROMOTION" "TARGET_CIR"
Стратегия, которая используется для кампании:
  • MAX_CLICKS — автостратегия для Поиска и рекомендаций;
  • TOP_MAX_CLICKS — автостратегия для Поиска;
  • TARGET_BIDS — средняя стоимость клика для Поиска и рекомендаций;
  • TOP_PROMOTION — вывод в топ;
  • TARGET_CIR — целевой расход.
autopilot ›
object (extcampaignCampaignAutopilotProperties)
Информация о кампании. Возвращается, если в параметре productAutopilotStrategy включена стратегия.
createdAt
string <date-time>
Дата создания кампании в формате RFC3339.
updatedAt
string <date-time>
Дата обновления кампании в формате RFC3339.
productCampaignMode
string
Default: "PRODUCT_CAMPAIGN_MODE_AUTO"
Enum: "PRODUCT_CAMPAIGN_MODE_AUTO" "PRODUCT_CAMPAIGN_MODE_MANUAL"
Режим создания и управления товарной рекламной кампанией:
  • PRODUCT_CAMPAIGN_MODE_AUTO — автоматически;
  • PRODUCT_CAMPAIGN_MODE_MANUAL — вручную.
